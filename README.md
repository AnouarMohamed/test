# Anouar Mohamed Portfolio

Personal portfolio website for Anouar Mohamed, focused on cloud-native DevSecOps, Kubernetes diagnostics, infrastructure automation, observability, and backend systems.

## Content

- Profile: [config/siteProfile.ts](config/siteProfile.ts)
- Resume: [components/Portfolio/Resume/Resume.tsx](components/Portfolio/Resume/Resume.tsx)
- Experience and skills: [components/Portfolio/Resume/Resume.config.tsx](components/Portfolio/Resume/Resume.config.tsx)
- Projects: [components/Portfolio/HighlightedProjects/HighlightedProjects.config.tsx](components/Portfolio/HighlightedProjects/HighlightedProjects.config.tsx)

## Stack

- Next.js
- React
- Redux
- Styled-components
- MongoDB

## Scripts

